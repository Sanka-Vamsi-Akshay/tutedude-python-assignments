# Assignment 5: Data Structures and Strings in Python


## Task 1: Create a Dictionary of Student Marks
- **File:** `task1.py`
- **Description:**  
  This program creates a dictionary where student names are used as keys and their marks as values.  
  It prompts the user to enter a student's name, then retrieves and displays the corresponding marks.  
  If the student is not found in the dictionary, it prints an appropriate message.

## Task 2: Demonstrate List Slicing
- **File:** `task2.py`
- **Description:**  
  This program:
  1. Creates a list of numbers from 1 to 10  
  2. Extracts the first five elements using list slicing  
  3. Reverses the extracted list  
  4. Displays both the sliced list and the reversed list  
  It demonstrates list slicing and basic list manipulation techniques in Python.
